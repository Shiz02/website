Переменные

Давайте представим, что нам нужно узнать за сколько машина из пункта A доедет в пункт B. Для того, чтобы сделать какие-то вычисления и найти результат, нам необходимы некоторые данные, такие как расстояние, скорость или время. Так же и в программировании. Чтобы нам делать какие-либо операции, нам нужны так называемые операнды, над которыми будут проводиться наши действия. Допустим, что мы знаешь расстояние, равное 240 киллометрам. Но ведь эта информация должна храниться где-нибудь, чтобы это знали не только мы, но и программа. Для этого в программировании есть такое понятие, как переменная:

Переменная - это определённая ячейка в памяти компьютера, которая хранит в себе какие-то данные определённого типа.

Переменная объявляется следующим образом:

  1. Тип Имя = Значение; //Процесс присваивания значения переменной при её объявлении, называется инициализацией
  2. string MyLoveGame; // Или значения может не быть вовсе
  3. MyLoveGame = "Minecraft"; // Но мы можем присвоить значения позже

Как вы уже поняли, чтобы присвоить переменной какое-либо значение используется оператор присваивания, а записывается одним знаком равно =.

Имя переменной можно задать любое, только нужно следовать некоторым требованиям:

  • Имя переменной может состоять из любых цифры и буквы, а так же символа подчеркивания, однако начинаться имя переменной должно либо с букв, либо со знака подчёркивания
  • В имени переменной не может быть знаков пунктуации и пробелов
  • Имя переменной не должно совпадать с зарезервированными словами C#-па. Например, тип данных:
  1. int int; // Так делать нельзя: программа выдаст нам ошибку

C# - Это регистрозависимый язык, а это значит, что одинаковые имена переменных по значению, но разные по написанию, будут воспринематься как две разных переменных:

  1. string Name = "Павел";
  2. string name;// Это две разных переменных!

В ходе программы мы не можем менять имя переменной(А это вовсе нам не нужно), за то мы можем менять значение этой переменной:

  1. int numOne;
  2. numOne = 4;
  3. numOne = 6;
  4. Console.Write("Переменная numOne равна: ");
  5. Console.Write(numOne); // Эта строка просто выводит переменную на консоль

Типы данных

Приведём ещё одну аналогию, например у нас есть две жидкости: обычная вода и бензин. К обоим этим, так скажем, объектам, можно привести общий тип "Жидкость". Но что если у нас разные вещи? Возьмём гитару и теслу, у этих вещей явно будут разные типы: "Музыкальный инструмент" и "Автомобиль". Так и в программировании, переменные бывают разных типов. C# - Это язык со статической типезацией, то есть каждая переменная должна быть с присвоенным ей типом, который объявляется перед именем переменной.

Объявим пару переменных разных типов:

  1. string Hi = "Привет!";
  2. char Simbol = 'd';
  3. int Years = 23;
  4. double S = 240.2;

Типы данных В C#, как и во многих других ЯП, есть своя система типов данных. Есть два вида типов данных: статические и динамические. C# относится к первым - это значит, что в C# нужно явно указывать тип данных перед именем переменной. Как мы уже говорили, тип данных определяет представление о виде объекта, который нужно записать в память компьютера, например: число или строка. В C# есть две подгруппы типов данных, сегодня мы поговорим о "примитивных типах":

  • int — Хранит в себе целое число от -2147483648 до 214748364. Поэтому у нас не получится записать в переменную число больше или меньше этого
  • uint — Если нам нужно записать число больше, чем нам может позволить тип int, то обращайтесь к этому челику. Этот тип может содержать от 0 до 4294967295. И да, этот тип не имеет возможности записать отрицательное число
  • long — Ну, а если вам и этого мало, то к вашему вниманию, этот тип хранит от –9 223 372 036 854 775 808 до 9 223 372 036 854 775 807
  • ulong — Как и в типе int, тип long не имеет отрицательного представления числа, за то он может вместить в себя от 0 до 18 446 744 073 709 551 615
  • short — Этот тип уже поменьше и хранит в себе от -32768 до 32767
  • ushort — Хранит от 0 до 65535
  • byte — Хранит от 0 до 255
  • sbyte — Хранит от -128 до 127
  • double — Тип чисел с плавающей точкой, который хранит в себе от ±5.0*10-324 до ±1.7*10308
  • float — Хранит числа с плавающей точкой от -3.4*1038 до 3.4*1038
  • decimal — Хранит от ±1.0*10-28 до ±7.9228*1028
  • char — Тип, который хранит в себе 1 символ, равный по величине в 2 байта
  • string — Хранит в себе набор символов
  • bool - Тип, который имеет два значения: true или false

Пока что некоторые типы вам могут быть не понятны, но бояться не за чем, ведь на данном моменте все они вам не понадобятся.

Возможно, у вас закрался вопрос. Зачем нужно столько типов данных? А возможно, что вы уже и сами нашли на него ответ. Могут возникнуть ситуации, при которых нам не понадобится работать с большими числами, тогда использовать byte вместо int, будет куда разумнее, ведь мы сэкономим место в памяти компьютера, и большие числа тоже требуются не всегда. Однако, в наше время этим можно пренебречь, темболее на настоящем этапе обучения.

Использование суффиксов

Попробуем объявить ещё пару переменных:

  1. float a = 3.14F;
  2. float b = 30.6f;
  3. decimal c = 1005.8M;
  4. decimal d = 334.8m;

Как видите, когда мы инициализируем переменную, мы добавляем суффикс к её значению - это нужно, чтобы компилятор мог отличить тип float или decimal от double, потому что на самом деле по умолчанию компилятор представляет любой тип с плавающей точкой, как тип double, для того, чтобы это обойти, нужно явно указать компилятору, что это другой тип, используя суффиксы: "F" или "f" для float, "M" или "m" для decimal.

Неявная типизация

Нам не обязательно указывать тип для переменной, компилятор сам может определить тип переменной смотря на её значение, однако лучше явно указывать тип
